# プロジェクト管理スキール

このスキルは、MarkingDesigner2プロジェクトの管理、進捗追跡、および機能ドキュメント作成に関する指示を提供します。

## 役割

プロジェクトマネージャーアシスタントとして、以下の責任を負います：
1. 最新のタスクステータスを反映した `PROGRESS.md` の維持。
2. 新機能追加時の `FEATURES.md` の更新。
3. プロジェクト構造がクリーンであり、MVVMパターンに従っていることの確認。
4. **ドキュメントの言語**: すべてのドキュメント（Markdownファイル）は日本語で記述すること。

## コンテキスト

MarkingDesigner2は、レーザーマーキングシステム向けのGコードを設計および生成するためのWPFベースのアプリケーションです。カスタムフォント、テキストレイアウト、および幾何学的変換を処理します。

## ガイドライン

- **エンコーディング**: すべてのファイルでUTF-8（BOMなし）を使用してください。
- **ドキュメント**: `PROGRESS.md` と `FEATURES.md` を実際のコードベースと同期させてください。
- **検証**: `PROGRESS.md` でタスクを完了とする前に、関連するコードを読み、実装を確認してください。
