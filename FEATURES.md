# 機能一覧 - MarkingDesigner2

このファイルでは、MarkingDesigner2プロジェクトの最新機能と機能を追跡します。

## コアマーキング機能

### Gコード解析とユーティリティ
- [x] **Gコード解析**: G00（高速位置決め）、G01（直線補間）、G02（円弧補間 CW）、G03（円弧補間 CCW）をサポート。
- [x] **幾何学的境界計算**: ワークスペース検証のため、Gコードシーケンスのバウンディングボックスを計算可能。

### フォントとテキストレイアウト
- [x] **カスタムマーキングフォント**: リソースファイルからの専用マーキングフォントの処理システム。
- [x] **テキスト変換**:
    - **ピッチ制御**: 文字間隔のための独立したXおよびYピッチ設定。
    - **回転**: 0, 45, 90, 135, 180, -45, -90, -135度の方向をサポート。
    - **文字高さ**: スケーラブルなフォントサイズ。
- [x] **境界計算**: 回転とピッチを考慮したテキスト文字列レイアウト境界の正確な計算。

### プロジェクト構成
- [x] **ジョブとシーケンス管理**: タスクは複数のシーケンスを含むジョブに整理。
- [x] **MVVMアーキテクチャ**: UIロジック（WPF）とビジネスロジック（ViewModels/Models）の明確な分離。

### 特殊マーキング（開発中）
- [ ] **円弧マーキング**: 円弧軌道に沿ったテキストのレイアウト。（ViewModelsに一部実装済み）
- [ ] **カレンダーマーキング**: 日付と時刻に基づいた動的なテキスト生成。（ViewModelsに一部実装済み）
- [ ] **増分マーキング**: 様々なフォーマットオプション（ゼロ埋め、数値など）を備えた自動シリアル番号生成。

## UI/UX
- [x] **デバイス範囲検証**: マーキングデータがデバイスの物理的な範囲を超えた場合の視覚的なフィードバック。
- [x] **軌道選択**: 直線およびその他の軌道のサポート。
